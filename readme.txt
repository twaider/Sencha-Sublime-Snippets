How to use
===========

Open in Sublime Text 2 and browse through packages:
Sublime Text 2 > File > Preferences > Browse packages.
Unzip the sencha folder in  Packages > User

=== Or use GIT on Mac OSX:

cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages && git clone git://github.com/savelee/Sencha-Sublime-Snippets.git "Sencha"
