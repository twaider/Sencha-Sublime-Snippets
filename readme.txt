How to use
===========

You can install packages with Sublime Package Control:
http://wbond.net/sublime_packages/package_control

Open in Sublime Text 2 and browse through packages:
Sublime Text 2 > File > Preferences > Browse packages.
Unzip the sencha folder in  Packages > User

=== Or use GIT on Mac OSX:

For example:
cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages && git clone git://github.com/savelee/Sencha-Sublime-Snippets.git "Sencha"

The following commands are available when opening a JavaScript file.
(make sure the files are saved as .js file with a correct name)

- extmodel <tab> (model)
- extcharts <tab> (pie chart)
- extchartscart <tab> (cartesian chart)
- extcontroller <tab> (controller)
- extclass <tab> (simple class)
- exteach <tab> (for each loop)
- extstore <tab> (simple store)
- extview <tab> (component view)
- extform <tab> (example form panel)
- exttpl <tab> (example xtemplate)
- extjsonp <tab> (Ext.data.JsonP.request)

- stmodel <tab> (touch model)
- ststore <tab> (touch store)
- stview <tab> (touch simple container)
- stcontroller <tab> (touch controller)
- stclass <tab> (define custom class)

- log <tab> (console.log)
